# my-pi-extensions

Personal pi extensions for [@earendil-works/pi-coding-agent](https://www.npmjs.com/package/@earendil-works/pi-coding-agent).

Each subdirectory is an independent pi extension package. Install one with:

```bash
cd <extension>
npm install
pi install /absolute/path/to/<extension>
```

## Extensions

- **[notify/](./notify)** — GNOME desktop notifications with tmux click-to-jump.
